Quienes Somos


Mercaya nació como una tienda de barrio en la ciudad de cartago que se dedica a la venta de productos de limpieza, comida y algunos enceres, con el objetivo de que los clientes puedan disfrutar de una experiencia familiar y de una compra segura.
Hace varios años la tienda se convirtió en una de las más importantes en la ciudad de Cartago, y pasamos a ser un supermercado y decidimos dar el paso a la web para competrir con los demás supermercados de la ciudad ofreciendo una experiencia online y domicilio gratis.




Acerca de la página



Este es el proyecto final de arquitectura de software, realizado por Juan Sebastián Vélez y entregado al profesor Carlos Alberto Ocampo.
Se trata de un crud de una tienda online, con una interfaz de usuario sencilla y una base de datos para almacenar los datos de los clientes y productos.
El objetivo de este proyecto es que los usuarios puedan disfrutar de una experiencia de compra online y que su domicilio llegue a su casa
El proyecto se realizó con el lenguage de programación python para el crud, se utilizó la librería Flask para el desarrollo de la aplicación web y se utilizó la librería SQLAlchemy para la base de datos.
